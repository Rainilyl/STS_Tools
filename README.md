# STS_Tools

![工具界面](Example.PNG)  
*语音转文本转语音工具界面*

## 📖 项目概述
**STS_Tools** 是一个正在开发中的多功能语音处理工具。

- **用途**:
  - 语音转录与转换
  - 会议记录自动化
  - 无障碍辅助工具开发

- **核心功能**:
  ✅ 已实现:
  - 语音转文本(STT)
  - 通过IndexTTS实现的文本转语音(TTS)
  
  🔜 规划中:
  - 屏幕OCR文字输入
  - 会议音频自动处理
  - 多TTS引擎支持

## 🚀 快速开始

### 环境要求
- Python 3.8+
- 音频输入设备
- 音频输出设备

### 运行步骤
```bash
git clone https://github.com/your-repo/STS_Tools.git
cd STS_Tools
pip install -r requirements.txt
启动indextts
python main.py
